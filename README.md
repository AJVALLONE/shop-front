# Shop Front — E-Commerce Product Page

A fully functional e-commerce product browsing and cart experience built in vanilla JavaScript.

## Features
- 8 products across 4 categories: Tees, Hoodies, Accessories, Outerwear
- Live category filtering and price sorting — no page reload
- Slide-in cart drawer with add, remove, and quantity controls
- Real-time cart total calculation
- Animated cart count badge
- Toast notifications on add-to-cart
- Fully responsive — mobile and desktop

## Tech Stack
HTML5 · CSS3 · JavaScript (ES6+) · DOM Manipulation
